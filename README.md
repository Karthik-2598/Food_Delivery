# Food Delivery App - Frontend

A modern, responsive, and feature-rich food delivery web application built with React and Vite. This frontend provides a premium user experience with dark mode, real-time notifications, cart management, checkout with location pinning, and separate customer/admin dashboards.


## Features

- **User Authentication**: Secure login/register with role-based access (customer/admin)
- **Browse & Cart**: Beautiful food list with search, filters, add to cart, and confetti animations
- **Checkout**: Address input with Google Maps location pinning and reverse geocoding
- **Order History**: View past orders with status tracking
- **Admin Dashboard**: Manage all orders, update status in real-time
- **Real-Time Notifications**: Socket.io powered updates for new orders and status changes
- **Dark Mode**: Full dark/light mode toggle with persistent preference
- **Responsive Design**: Mobile-first, works perfectly on phones, tablets, and desktops
- **Premium UI**: Glassmorphism cards, gradients, animations (Framer Motion), and Poppins typography

## Tech Stack

- **Framework**: React + Vite
- **UI Library**: Material-UI (MUI)
- **Routing**: React Router
- **State Management**: React Context
- **Animations**: Framer Motion
- **Maps**: Google Maps API + React Google Maps
- **Notifications**: React Toastify
- **Real-Time**: Socket.io Client
- **HTTP Client**: Axios
- **Styling**: MUI sx props + custom CSS
